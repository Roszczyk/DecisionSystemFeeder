# LAYER 1 of Decision system

```Purpose```: Data acquisition

## Components

Data are acquired using following sources:
* Computer Vision 
    * Image-based object detection using RGB camera ([resources](CV_RGBCamera))
    * Thermal image-based object detection using RGB camera infrared sensor ([resources](CV_InfraredCamera))
    * Sound-based event detection
* Internet of Things Sensors
    * Temperature
    * Light intensity
    * Presence and motion
    * Force - Tensometer
